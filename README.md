使用方法 / How to Use

为了提供最好的体验，我已经在打包时内置了完整的运行环境。
For the best experience, the application comes with a pre-packaged runtime environment.

    获取软件 (Get the App):

        Go to the Releases page on the right.

        Download the latest ZIP file (e.g., ChoicePrison_v1.0.zip).

    直接运行 (Run):

        解压下载得到的压缩包。 (Unzip the downloaded file.)

        双击运行其中的 .exe 文件 即可开启性格探索之旅。 (Double-click the .exe file to start.)

        无需安装任何 Java 环境！ (No Java environment installation required!)
